# markdown's tools

>這是一個註解說明。

```md

>這是一個註解說明。
```

>[!tip]
>這是個提示建議。

```md
>[!tip]
>這是個提示建議。
```

>[!important]
>這是非常重要的說明。

```md
>[!important]
>這是非常重要的說明。
```

>[!warning]
>這是警告資訊！

```md
>[!warning]
>這是警告資訊！
```

>[!caution]
>這是小心提醒。

```md
>[!caution]
>這是小心提醒。
```

>[!example]
>這是一個範例。

```md
>[!example]
>這是一個範例。
```

>這是一段引用。

```md
>[!quote]
>這是一段引用。
```

>[!info]
>這是一般資訊。

```md
>[!info]
>這是一般資訊。
```
