# Fermat’s Little Theorem 小費馬定理

## 使用時機 與 條件

>[!note] 
>## 使用時機
> 當要將兩個數字快速相除且取餘時，因為 (a/b)%m != (a%m) / (b%m)，故需要


- 條件: p 為質數 且 a % p != 0　

那麼

$`a^{p−1} ≡ 1 (mod p)`$

然後可以推導出以下:

$`a^(p−2) ≡ a^{−1} (mod p)`$

那麼當我們要解以下方程式時

$`(a / b) % p`$

$`(a % p) * (a^{mod-2})`$

這樣就可以了!!!!
